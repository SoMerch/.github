# Code of Conduct

SoMerch public repositories are intended to support clear, professional, and useful collaboration around documentation, public roadmap feedback, integration concepts, and product information.

We expect all participants to communicate respectfully and constructively.

---

## Expected behavior

Participants are expected to:

- be respectful and professional
- stay on topic
- provide useful context
- give clear feedback
- avoid personal attacks
- respect privacy and confidentiality
- avoid sharing sensitive customer or employee data

---

## Unacceptable behavior

Unacceptable behavior includes:

- harassment or abusive language
- discrimination or personal attacks
- spam or unrelated promotion
- sharing private or sensitive data
- publishing credentials, tokens, or secrets
- impersonation or misleading claims
- attempts to use public repositories for support cases involving private customer data

---

## Scope

This code of conduct applies to:

- GitHub issues
- pull requests
- discussions
- public comments
- documentation contributions
- public roadmap feedback

---

## Enforcement

SoMerch may remove comments, close issues, restrict participation, or block accounts that violate these guidelines.

For urgent concerns, contact:

[support@somerch.co](mailto:support@somerch.co)
