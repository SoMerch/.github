# SoMerch GitHub Profile

This repository contains organization-level public files for the SoMerch GitHub organization.

It includes:

- organization profile README
- support policy
- security reporting instructions
- code of conduct
- contribution guidance
- default issue templates

These files help keep SoMerch public repositories consistent and easier to use.

---

## Organization profile

The public GitHub organization profile is located at:

```text
profile/README.md
```
